
### Prerequisites:
- Node.jS (v16 or later) installed on your machine.

- ### Running the Project locally:
1. Clone the repository
2. Navigate to the project directory (cd project_Name)
3. npm install
4. npm run dev
5. Open it in the browser, http://localhost:3000
