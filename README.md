
### Prerequisites:
- Node.jS (v16 or later) installed on your machine.

- ### Running the Project locally:
- Clone the repository
- Navigate to the project directory (cd project_Name)
- npm install
- npm run dev
- Open it in the browser, http://localhost:3000
